# DEX Frontend - React

This is the React-based user interface for the Decentralized Exchange (DEX). It allows users to connect their MetaMask wallets, swap tokens, and manage liquidity pools.

## 🚀 Features

- **Wallet Connection**: Supports MetaMask and other EIP-1193 compatible wallets.
- **Token Swaps**: Easy-to-use interface for swapping between supported tokens.
- **Liquidity Management**: Add or remove liquidity from existing pools.
- **Real-time Data**: Displays current reserves, prices, and user balances.
- **Responsive Design**: Optimized for both desktop and mobile viewing.

## 🛠️ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- A running Hardhat node (for local development)

### Installation

```bash
cd frontend/frontend-react
npm install
```

### Development

To start the development server:

```bash
npm start
```

The application will be available at `http://localhost:3000`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `build` folder.

## 🧪 Testing

To run the test suite:

```bash
npm test
```
